Class: PasswordInput

candystore. PasswordInput

The PasswordInput extends TextInput with the option that its input type will be set to 'password'. Supports revealing and obscuring the entered password. Also delegates surrogate to Input: instantiating an Input with 'type'='password' will yield a PasswordInput instance.
