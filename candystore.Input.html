Class: Input

candystore. Input

The Input is the base class for all input widgets: text, checkbox, radio button, etc. Inputs can be validated by supplying a validator function.

validatorFunction :function

Function that validates the input's value. Receives the input value as argument and is expected to return undefined when it's valid, any other value when it's not. Return value will be stored as instance property (lastValidationError) as well as passed to the validity event as payload.
